How Catch works

Real context before the chat ever starts.

The whole point is simple: everyone shares the same club, the same route, and the same early alarm before anyone gets a match screen.

  1. 01

    Book a group run

    Follow clubs, browse upcoming runs, and pick one that matches your city, distance, and vibe.

  2. 02

    Actually show up

    Run with the club. No swiping happens during the event. Catch is built around the real-world moment, not around the pre-game.

  3. 03

    Swipe within 24 hours

    After the run, the roster unlocks for one day. Catch anyone who genuinely caught your eye.

  4. 04

    Match and message

    If they catch you back, the chat opens with shared context already baked in. No blank-slate small talk required.

Why Catch feels different

Discovery, commitment, then the catch. In that order.
