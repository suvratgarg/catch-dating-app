## Use Policy

- These screenshots are references and evidence only. Do not copy competitor
  visuals, screenshots, logos, or photography into Catch production assets.
- Production Catch assets should use owned media, app-generated screenshots, or
  explicitly approved generated imagery.
- If a screenshot is blocked or partial, rely on the research doc's text notes
  and the source URL instead of treating the image as visual truth.
