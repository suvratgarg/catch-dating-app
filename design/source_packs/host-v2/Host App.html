The consolidated host app. Four club-shaped tabs became four lifecycle-shaped ones — Today · Events · Inbox · Organizer — with Manage and Insights as first-class destinations. Every screen is a live template composed purely of catalogue widgets; the create / edit / sheet flows are re-homed from the on-system rebuild so nothing is lost.

The app bar · four lifecycle tabs

The new shell. Today is the time-aware ops home (live hero + triage queue); Events owns the create + lifecycle list; Inbox is event-scoped with blasts; Organizer merges the old Clubs brand + Account profile into one durable surface.

01TodayTabOpen
NextUpHero (live countdown) + NeedsYouQueue (approve / offer / reply, inline) + the week's events.
02EventsTabOpen
New event + Repeat-last, an Upcoming / Live / Past SegPill, month-grouped EventLifecycleRows. Tap the segments.
03InboxTabOpen
Booked / Prospective threads, event-scoped. Tap the blast row for the BlastComposer sheet.
04OrganizerTabOpen
OrganizerHeader + payouts nudge + brand stats + team + a TrendStrip into Insights + manage rows.

The per-event hub & analytics

Reached from a Today hero, an Events row, or a NeedsYou item. Manage opens on the event's lifecycle state; Insights is the cross-event analytics destination from the Organizer's trends.

05Manage eventSetup · Guests · Live · ReportOpen
One screen, four modes — MetricGrid, RosterBoard, the LiveConsole run-of-show + big check-in, and the post-event report. Tap the modes.
06InsightsDestinationOpen
DateRangePicker (tap the range chip), ▲▼ deltas, the demand→booking trend, a by-format split, top events, and data-quality fixes.

Create & edit flowsRe-homed

The deeper wizards, preserved from the on-system rebuild and re-homed under the new IA — create lives in Events, edit in Manage, the organizer wizards in Organizer. These prevent the prototype's toast-stub regressions (R1–R3).

07Create eventEvents › NewOpen
The 5-step wizard, reached from Events → New event (and Repeat-last). Tap Next to step through.
08Event createdConfirmOpen
The confirmation after scheduling. Tweak adds the invite-only code + private link.
09Edit eventManage › SetupOpen
Reached from Manage → Setup → Edit. Tweak "Has bookings" locks schedule + policy to read-only.
10Create organizerOrganizer › NewOpen
The organizer (club) wizard — identity, contact, host + event-success defaults. Tap Next.
11Edit organizerOrganizer › EditOpen
The wizard flattened — cover photos + Identity / Contact / Event-defaults rows.

The modal layers behind the screens — add a co-host, resume a draft, the Stripe payouts hand-off, and the confirm dialogs. Preserved from the on-system rebuild (regression R4); toggle Tweaks on the dialogs to switch which one.
